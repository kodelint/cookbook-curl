Description
============

Installs/Configures curl

Requirements
============

* Chef 12
* Ruby 2.1.0/2.2.0

Attributes
==========

* `default['curl']['libcurl_packages']` - A list of libcurl packages to install.

Usage
=====

```json
"run_list": [
    "recipe[curl]"
]
```

default
-------

Installs/Configures curl

libcurl
-------

```json
"run_list": [
    "recipe[curl::libcurl]"
]
```

Install/Configure libcurl packages
